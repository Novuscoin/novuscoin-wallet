En utilisant cette boîte de dialogue, vous pouvez recevoir des novuscoins (vous devez avoir installé novuscoin Wallet).

D'abord, saisissez le montant de novuscoins à recevoir.

Ensuite, l'autre personne doit scanner le QR code généré. Ou bien, vous pouvez envoyer une URL novuscoin à une personne à distance avec le bouton "publier" dans la barre d'action. L'autre personne devra cliquer l'URL novuscoin.

Dasn les deux cas, l'autre personne se verra présenter une boîte de dialogue pour envoyer des novuscoins avec tous les champs pré-remplis. Si elle accepte, elle n'a plus qu'à envoyer la transaction.
